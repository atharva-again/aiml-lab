Simple repo to handle by AIML Lab Assignments / Projects

# Steps To Use This Repo

## Clone It

```
git clone https://github.com/atharva-again/aiml-lab.git
```

## Download uv (if not already in your system)

See this: https://docs.astral.sh/uv/#highlights

## Sync dependencies

Run this command:

```
uv sync
```

You are good to go now!

# Repo Structure

The repo is divided into sub-folders. One folder per assignment and all its related artifacts.
